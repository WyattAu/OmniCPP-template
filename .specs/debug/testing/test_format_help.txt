usage: OmniCppController.py format [-h] [--files [FILES ...]]
                                   [--directories [DIRECTORIES ...]] [--check]
                                   [--dry-run] [--cpp-only] [--python-only]

options:
  -h, --help            show this help message and exit
  --files [FILES ...]   Specific files to format
  --directories [DIRECTORIES ...]
                        Directories to scan for files
  --check               Only check formatting without modifying files
  --dry-run             Run in dry-run mode
  --cpp-only            Only format C++ files
  --python-only         Only format Python files
